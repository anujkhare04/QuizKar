
# QuizKar - Full Stack Quiz Platform

## Overview

QuizKar is a modern full-stack quiz platform designed to help users test their knowledge, track performance, and improve learning through detailed analytics. The platform provides multiple quiz modes, leaderboard rankings, performance insights, and AI-powered suggestions for continuous improvement.



## Features

### Authentication & User Management

* User Registration and Login
* JWT-Based Authentication
* Secure User Sessions
* User Profile Management

### Quiz System

* Attempt Quizzes Across Multiple Categories
* Multiple Difficulty Levels
* Randomized Questions
* Instant Result Evaluation

### Quiz Modes

* Timed Mode
* Stop on Incorrect Mode
* Number of Questions Mode

### Performance Analytics

* Score Trend Analysis
* Accuracy Tracking
* Performance Distribution
* Improvement Detection
* Learning Progress Monitoring
* Streak Tracking

### Leaderboard

* Global Ranking System
* User Score Comparison
* Competitive Learning Environment

### AI Features

* AI-Powered Learning Suggestions
* Personalized Improvement Recommendations

## Installation

### Clone Repository


git clone https://github.com/yourusername/QuizKar.git


### Frontend Setup


cd client
npm install
npm run dev


### Backend Setup


cd server
npm install
npm run dev


## Environment Variables

Create a `.env` file inside the server directory.

env

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_secret_key
